# 🧱 AI-Driven Concrete Mix Optimization (MixXperts)

An AI-powered concrete mix design optimization framework using Artificial Neural Networks (ANN), XGBoost, and the NSGA-II genetic algorithm. Designed to balance compressive strength, material cost, and sustainability parameters.

### 🏆 Achievements
- **Winner** — TECH TONIC Software Hackathon, NIT Puducherry
- **Impact** — Reduces trial batches by up to 60%, significantly improving cost efficiency and reducing material waste.

### ✨ Features
- **Machine Learning Models:** Trained on extensive concrete mixture datasets using ANN and XGBoost.
- **Multi-Objective Optimization:** Uses NSGA-II to find the Pareto optimal front balancing strength, cost, and embodied carbon.
- **Web UI:** Interactive Flask-based dashboard for real-time mix evaluation and prediction.
- **PDF Report Generation:** Automated mix design reports.

### 🛠️ Tech Stack
- Python, Flask, PyTorch / TensorFlow, XGBoost, Scikit-Learn
- NSGA-II Algorithm
- HTML/CSS/JS (Frontend)
